
# flask-whisper

## Install

```
$ python3 -m venv .
$ source ./bin/activate
$ pip3 install flask-sock 
$ pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
$ pip3 install openai-whisper
```

## Run

```
$ python3 app.py
```
